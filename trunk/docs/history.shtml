History

openADAMS Devel

New features

  • Text editor now supports links (Menu Edit/Edit link, Edit/Delete Link, ...)
  • Text editor has new menu item to insert plain text
  • Support for opening database files by drag'n drop
  • Command line switches '-DEBUG', '-INFO', '-ERROR' for different log levels
  • oareport.xsl now includes oareport.css for screen output and oaprintreport.css for print output
  • Export to Compressed Help Format (using Microsoft Help Compiler), either from main window or using script naf_exportchm.py

Changes

  • None

Bug fixes

  • oareport.xsl: removed whitespaces from links

Known issues

  • oaeditor: playing extensively with drag and drop in the artifact tree may cause a crash of the application
  • Documentation is missing
  • oaxml2db.py: experimental
